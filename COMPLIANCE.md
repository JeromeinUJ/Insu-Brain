# ⚠️ 컴플라이언스 가이드 (법적 준수사항)

## 왜 이게 중요한가요?

보험업법 제97조에 따라 **금융감독원 심의를 받지 않은 광고물을 공개 게시**하면:

- 💰 **과태료 최대 3천만원**
- 📋 **설계사 자격 정지 가능**
- 🏢 **회사 영업 제재**

→ **Insu-Brain은 심의 미필 자료이므로, 법적 보호장치가 필수입니다.**

---

## 🛡️ Insu-Brain의 3단계 방어 시스템

### 1단계: 강제 워터마크 (`lib/utils.ts`)

모든 이미지 저장 시 자동으로 다음 요소가 삽입됩니다:

#### a) 하단 Disclaimer Bar
```
⚠️ 본 자료는 심의필이 없는 내부 교육용 자료이며, SNS 게시를 금합니다.
1:1 상담 목적으로만 활용 가능합니다.
```

- 위치: 이미지 최하단
- 색상: 회색 배경 (#4a4a4a) + KB Yellow 상단 테두리
- 글자: 흰색, 11px, 굵게

#### b) 대각선 워터마크
```
INTERNAL USE ONLY
```

- 위치: 이미지 중앙
- 각도: -45도 대각선
- 색상: KB Yellow 반투명 (opacity 15%)
- 크기: 48px

**코드 위치:** `lib/utils.ts` → `exportWithWatermark()` 함수

---

### 2단계: 법적 경고 모달 (`components/compliance-alert.tsx`)

[이미지 저장] 버튼 클릭 시 **필수 확인 절차:**

#### 경고 내용
1. **심의 미필 명시**
   - "본 자료는 금융감독원 심의를 받지 않은 내부 교육용 자료입니다."

2. **과태료 위험 고지**
   - "SNS, 카페, 블로그 등 공개 게시 시 보험업법 제97조에 따라 **과태료 3천만원 이하** 처분 대상이 됩니다."

3. **허용 사용처 안내**
   - ✅ 카카오톡, 문자 등 **1:1 개별 전송**
   - ❌ SNS, 카페, 블로그 등 **공개 게시**

#### 동의 절차
- 체크박스 클릭: "위 내용을 충분히 이해했으며, 1:1 전송 용도로만 사용하겠습니다."
- **동의하지 않으면 저장 불가** (버튼 비활성화)

**코드 위치:** `components/compliance-alert.tsx` → `ComplianceAlert` 컴포넌트

---

### 3단계: UI/UX 상 지속적 리마인더

#### 메인 페이지 하단 안내

```
⚠️ 법적 준수사항:

SNS 게시 시 과태료 대상이 되므로, 반드시 1:1 상담 목적으로만 사용하세요.
```

```
카카오톡이나 문자로 개별 전송하세요. (SNS 업로드 금지)
```

**코드 위치:** `components/recommendation-tab.tsx` → Results Section

## 📝 실제 사용 시나리오

### ✅ 허용되는 사용 (OK)

#### 1. 카카오톡 1:1 전송
1. Insu-Brain에서 비교 분석 수행
2. [이미지 저장] 클릭 → 경고 확인 → 동의
3. 저장된 이미지를 카카오톡에서 **특정 고객 한 명**에게 전송
4. 메시지 예시: "고객님, 제가 준비한 맞춤 분석 자료입니다. 확인해보시고 궁금하신 점 연락주세요!"

#### 2. 문자 메시지 전송
1. 이미지 저장 후 MMS로 개별 전송
2. 전화 상담과 함께 병행 가능

#### 3. 대면 상담 시 태블릿 화면 직접 보여주기
- 이미지 저장 없이 화면 자체를 고객에게 보여주는 것도 OK

---

### ❌ 절대 금지 (과태료 위험)

#### 1. 인스타그램/페이스북 스토리 업로드
- "좋은 정보 공유합니다~" → **즉시 적발 대상**
- 워터마크가 있어도 **소용없음** (심의 미필이 명시되어 있으므로 더 위험)

#### 2. 네이버 카페/블로그 게시
- 보험 정보 공유 카페에 올리는 것도 **불법**
- "비공개 카페"라도 불특정 다수가 볼 수 있으면 금지

#### 3. 단체 카카오톡방 공유
- "설계사 모임" 같은 단체방도 **위험 지대**
- 1:1 대화방만 허용

## 🔧 개발자를 위한 커스터마이징

### 워터마크 문구 변경

`lib/utils.ts` 파일의 `exportWithWatermark` 함수에서:

```typescript
disclaimer.innerHTML = `
  ⚠️ 본 자료는 심의필이 없는 내부 교육용 자료이며, SNS 게시를 금합니다.<br/>
  1:1 상담 목적으로만 활용 가능합니다.
`;
```

→ 회사 정책에 맞게 문구 수정 가능

### 워터마크 투명도 조절

```typescript
watermark.style.cssText = `
  ...
  color: rgba(255, 188, 0, 0.15); // 0.15 → 0.3으로 변경하면 더 진하게
  ...
`;
```

### 경고 모달 텍스트 변경

`components/compliance-alert.tsx` 파일에서:

```tsx
<DialogDescription>
  <p>본 자료는 금융감독원 심의를 받지 않은 내부 교육용 자료입니다.</p>
  {/* 여기 텍스트 수정 */}
</DialogDescription>
```

## 🚨 만약 실수로 SNS에 올렸다면?

1. **즉시 삭제** (1분 내 삭제해도 캡처 가능하므로 주의)
2. **회사 컴플라이언스팀 보고**
3. **사후 대응 매뉴얼 따르기**

### 예방책

- 이미지 저장 시 **파일명에 경고 포함**:
  ```
  kb-comparison-1대1전송용-SNS금지.png
  ```
