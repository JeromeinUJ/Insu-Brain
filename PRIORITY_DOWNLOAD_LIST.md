## 📝 다운로드 가이드

### Step 1: 다운로드 (수동)
```
1. 위 표의 "다운로드 URL" 또는 "검색 URL" 클릭
2. 보험사 공시실에서 해당 상품 찾기
3. "약관보기" 또는 "약관 다운로드" 클릭
4. PDF 저장
```

### Step 2: 파일명 규칙
```
{회사명}_{상품명}.pdf

✅ 올바른 예시:
- KB손해보험_간편건강보험.pdf
- 삼성화재_암보험.pdf
- 현대해상_운전자보험.pdf

❌ 잘못된 예시:
- KB 간편건강보험.pdf (공백)
- kb-insurance.pdf (영문)
- 보험약관.pdf (정보 없음)
```

### Step 3: 저장 위치
```bash
Insu-Brain/pdfs/
```

### Step 4: 업로드
```bash
# Supabase Storage 버킷 생성 (최초 1회)
# Supabase Dashboard → Storage → New Bucket
# Name: insurance-pdfs, Public: ON

# PDF 업로드
node upload-pdfs.js
```

## 🔍 다운로드 방법 (상세)

### A. KB 손해보험
1. https://www.kbinsure.co.kr 접속
2. 하단 "상품·약관" 또는 "공시실" 클릭
3. 상품명 검색 (예: "간편건강보험")
4. 상품 선택 → "약관보기" 클릭
5. PDF 다운로드

### B. 금융감독원 통합공시
1. https://fine.fss.or.kr 접속
2. "보험" 메뉴 선택
3. 보험사 → KB손해보험 선택
4. 상품명 검색
5. 약관 다운로드

### C. 구글 검색 활용
1. 검색어: `{보험사명} {상품명} 약관 PDF`
2. 예: "KB손해보험 간편건강보험 약관 PDF"
3. 공식 사이트 결과 클릭
4. PDF 다운로드

## 📞 문제 해결

### Q: PDF를 찾을 수 없어요
**A:** 다음 방법 시도
1. 보험사 공식 홈페이지 → 공시실
2. 금융감독원 FINE 포털
3. 고객센터 문의 (전화/챗봇)

### Q: 약관이 여러 개예요
**A:** 다음 우선순위로 선택
1. "주계약 약관" (메인)
2. 최신 버전 (날짜 확인)
3. "일반형" 또는 "표준형"

### Q: 파일 크기가 너무 커요
**A:** 괜찮습니다!
- Supabase: 50MB까지 지원
- 대부분 약관: 3~10MB
